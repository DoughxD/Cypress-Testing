#my cypress automation

## 📋 About the Project

This project utilizes **Cypress** to automate the testing of a **login feature** and a **user update feature** on the OrangeHRM system.

Cypress is a JavaScript-based end-to-end testing framework that allows developers to write tests and execute them in a real browser, ensuring more reliable and accurate results.

## 🎯 Tested Features

- ✅ Successful login
- ✅ Login with invalid credentials
- ✅ User data update
- ✅ Required field validation

## 🛠️ Technologies Used

- **Cypress** (testing framework)
- **JavaScript** (programming language)
- **OrangeHRM** (application under test)

## 🚀 How to Run

```bash
# Install dependencies
npm install

# Open Cypress interface
npx cypress open

# Run tests in headless mode
npx cypress run
